# Crime-Rates-in-Singapore-2022---2024-
This is a python coding project that involves analysis of Crime Rates in Singapore from 2020-2021

# Dataset used
[https://tablebuilder.singstat.gov.sg/table/TS/M891481]
Source: SingStat


# About the Project 
This project involves using python code to transform and analyse Crime Rate data in Singapore fro 2020-2024 

# How to run the project 

1. Open the notebook in Google Colab:
   - Upload `SG Crime Rates (2020-2024).ipynb` to Colab  

2. Upload the downloaded dataset from repository into Colab 

3. Run all cells from top to bottom.
